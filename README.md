## ğŸ¯ Game Mechanics

- Slide the tiles in the direction of arrow keys
- Two tiles with the same value merge into one with double the value
- A new tile (Flour or Egg) is spawned after each move
- When no valid moves remain (grid is full), game over
- To win, create the Pie tile (2048)
